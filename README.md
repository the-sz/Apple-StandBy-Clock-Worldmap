# Apple StandBy Clock Worldmap

Create a HTML page with a dotted world map, your location highlighted and the current time. Like the Standby Clock on iOS devices.

![](.github/example.png)

## Worldmap Creation

* Edit `create.js` file
* Run node.js script:
```bash
npm install
node create.js
```
* open `index.html`
